Adding games:
To add a game add a folder to the games folder
Make a new include statement in main for the game
If you want your stats to be stored you must edit the readStats and writeStats functions under the user folder, and the data struct for user data in stats.h

More info on the readStats editing
edit the read amounts
and save to the edited struct
